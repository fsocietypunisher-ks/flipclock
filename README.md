# ⏰ Flip Clock Pro

A lightweight **flip clock with timer** featuring smooth flip animation and sound alerts.

## Requirements

* Linux (Ubuntu / Debian recommended)
* Python **3.6+**
* GTK3

Install dependencies:

```bash
sudo apt install python3-gi gir1.2-gtk-3.0
```

## Installation

Clone or download the project and make sure the file is present:

```
flipclockV1.py
```

## Run

```bash
python3 flipclockV1.py
```

This will open:

* ⏰ **Clock Window**
* ⏱️ **Timer Window**

## Run in Background

```bash
python3 flipclockV1.py &
```

## Stop the App

```bash
pkill -f flipclockV1
```
